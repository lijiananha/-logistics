## 物流查询

基于 [极速数据](https://www.jisuapi.com/) 物流查询，每天免费1000次

## 安装

```
composer require lijiananha/logistics -vvv
```


##配置
到[极速数据](https://www.jisuapi.com/) 申请appkey

config/service
```

'logistics' => [
    'key'   => 'xxxxxxxx'
]
```

## 使用

$number = '单号';
app('Logistics')->query($number)
